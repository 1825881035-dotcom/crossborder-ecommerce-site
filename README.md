# 跨境电商个人品牌主页

这是一个可直接打开的静态个人主页，适合用于跨境电商个人品牌、品牌出海顾问、Amazon/FBA 运营服务展示。

页面包含首屏定位、服务能力、出海路径、增长方法、运营看板、案例展示、合作套餐和联系方式。

## 文件

- `index.html`：页面结构和中文文案
- `styles.css`：页面样式和响应式布局
- `assets/hero-crossborder.png`：首屏视觉图
- `assets/service-research.png`：选品分析配图
- `assets/service-ads.png`：广告复盘配图
- `assets/service-fba.png`：FBA 供应链配图
- `assets/service-brand.png`：品牌内容配图
- `assets/consulting-session.png`：顾问沟通配图

## 修改建议

把以下占位内容替换成你的真实信息：

- `你的个人品牌`
- `hello@example.com`
- `YourWeChatID`
- 案例区里的真实项目、数据和结果

直接双击 `index.html` 即可在浏览器打开。
